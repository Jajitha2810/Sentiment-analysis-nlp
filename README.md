# Sentiment Analysis using Deep Learning for NLP

## Overview
A deep learning-based sentiment analysis system that classifies text as positive, negative, or neutral using LSTM and transformer-based architectures.

## Features
- Text preprocessing and tokenization
- LSTM-based sentiment classification
- Transformer model comparison (BERT)
- Visualization of results and model performance
- Trained on IMDB movie review dataset

## Technologies Used
- Python 3.11
- TensorFlow / Keras
- Hugging Face Transformers
- NLTK
- Matplotlib & Seaborn
- Pandas & NumPy
